Basic commands in Linux.


`pwd` : It shows present working directory 


```
morty@c137:~$ pwd
/home/morty
```


`cd` , `cd ~` : Change current directory to your home directory

```
morty@c137:~/Documents$ cd
morty@c137:~$
```

```
morty@c137:~/Documents$ cd ~
morty@c137:~$
```
